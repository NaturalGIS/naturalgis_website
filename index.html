NaturalGIS é uma empresa Portuguesa, activa em todo o mundo, que opera no sector dos Sistemas de Informação Geográfica. Somos especialistas no software SIG QGIS (Desktop, Server e Web), ao qual estamos directamente ligados, como parte da equipa que o desenvolve, assim como na Base de Dados de classe Entreprise PostgreSQL e sua extensão para os dados geográficos PostGIS.
