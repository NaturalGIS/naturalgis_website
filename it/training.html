Open Source GIS Training

Since established in 2009, NaturalGIS offers high quality training, using the latest available versions of Open Source GIS software. We give training with standard or customized contents, open to the public or at the client's location/venue. We specialize in: QGIS (Desktop, Server e Web), PostgreSQL/PostGIS and Python for QGIS (pyQGIS).

Some features of NaturalGIS training courses:

  • High quality: are regularly evaluated with very high popularity ratings (between 82% and 93%, based on anonymous forms)
  • Interactive: rich of practical exercises and with constant support by trainers
  • Direct contribution to software: Giovanni Manghi is QGIS main tester, and NaturalGIS supports software like QGIS, GRASS, PostGIS, GDAL, etc. donating part of training subscription fees to their development
  • Competitive price: lower than the equivalent training based on closed/proprietary software
  • Multilingual: We can provide training in Portuguese, English or Italian
  • Discounts: for students, unemployed people and previous clients

GIS analysis with QGIS

To learn how to do raster and vector GIS analysis, joining the power of tools like GRASS, SAGA, Orfeo Toolbox with the simplicity of use of QGIS. The course is aimed to who wants to learn how to use the GIS to do predictive analysis. Is adviced to have at least basic knowledge about SIG software. Is suggested to have also a basic about QGIS. Duration: Three days (24 hours). Trainer: Giovanni Manghi.

  1. Introduction to the software used in the course
  2. QGIS and its plugins
  3. Additional plugins
  4. QGIS Processing framework (was SEXTANTE)
  5. GRASS
  6. SAGA
  7. Orfeo ToolBox (OTB)
  8. Basic vector analyses: dissolve, merge, overlay, attribute management, etc.
  9. Correction of topological errors and vector generalization
  10. Geomorphology analyses: digital terrain modules, contour, slope, aspect, shaded maps
  11. Interpolations

  17. 3D visualization
  18. CMulticriteria analyses: priority and risk maps
  19. Unsupervised, automatic land use classification, feature extraction
  20. Modelling through graphical interface and command line; batch processing for automatic processing

Spatial databases

The Spatial databases course is aimed at those who need to learn how to manage a database of complex geographic data. The course also covers the use of PostGIS many database functions that allow for complex geoprocessing to be carried out within the database and displayed in real-time. It is suggested to have basic knowledge of the SQL language. Duration: Three days (24 hours). Trainer: Giovanni Manghi.

  6. Access control (users, groups and permissions)

  14. Advanced geoprocessing functions
  15. PostGIS rasters
  16. Topology support
  17. Backing up (dumping and restoring)

Programming Python for QGIS

The course teaches, step by step, how to develop a Python plugin for QGIS. Knowledge of basic Python skills is useful, but not necessary. At least a basic knowledge of programming principles and of QGIS is needed. Duration: Three days (24 hours). Trainer: Luigi Pirelli.

  • Calendar
  • Subscribe

Content:

  1. How to create a python plugin for QGIS
  2. the python syntax (indentation, colon, ...)
  3. my first python plugin: display a message box
  4. create GUIs using QtDesigner (.ui)
  5. second plugin: add a layer to QGIS canvas
  6. create a resource file (.qrc)
  7. The main QT API classes
  8. QTCore module (QObject, QString, QAction, QFile, ...)
  9. QTGui module (QWidget, QDialog, QMessageBox, QFileDialog, ...)
  10. access QT API through python: PyQT (QT types mapped by python native types, out parameter)
  11. The main QGis API classes and their relations
  12. constants and settings: QGis and QgsApplication classes
  13. plugin interface: QgisInterface and common methods used in plugins
  14. canvas, layers and legend: QgsMapCanvas, QgsMapLayer, QgsLegendInterface, QgsMapLayerRegistry
  15. vector layer: QgsVectorLayer, QgsVectorDataProvider, QgsField, QgsFeature, QgsGeometry
  16. raster layer: QgsRasterLayer, QgsRasterDataProvider
  17. map tools and rubber band: QgsMapTool, QgsMapToolEmitPoint, QgsRubberBand
  18. canvas items and snapper: QgsMapCanvasItem, QgsVertexMarker, QgsSnapper
  19. CRS classes: QgsCoordinateReferenceSystem, QgsCoordinateTransform
  20. util classes: QgsDataSourceURI
  21. Let’s use pyQGIS
  22. code examples
